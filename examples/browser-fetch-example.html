HLS Parser - Browser Fetch Example

Fetch HLS Playlist from URL

This example demonstrates using the HLS parser with Web Streams API to parse playlists fetched from a URL.

How it works

// In browser with bundled version:
import { HLS } from '../dist/index.browser.js';

// Or in Node.js/bundlers:
import { HLS } from 'streaming-hls-parser';

// Fetch HLS playlist from a URL
const response = await fetch(playlistUrl);
const text = await response.text();

// Parse the playlist (auto-detects media vs multivariant)
const playlist = HLS.parse(text);

// Convert back to HLS format
const hlsString = HLS.stringify(playlist);

HLS Parser - Browser Fetch Example

Fetch HLS Playlist from URL

This example demonstrates using the HLS parser with Web Streams API to parse playlists fetched from a URL.

How it works

import { HLS } from 'streaming-hls-parser';

// Fetch HLS playlist from a URL
const response = await fetch(playlistUrl);

// Parse the playlist using Web Streams
// The response.body is a ReadableStream of Uint8Array chunks
const playlist = await HLS.parseMediaPlaylist(response.body);

// Or parse a multivariant playlist
const playlist = await HLS.parseMultivariantPlaylist(response.body);

// Or use auto-detection
const playlist = await HLS.parse(await response.text());
